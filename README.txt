Sujet :

5/ Code secret
Le but de ce projet est de réaliser un programme permettant de crypter un message à l'aide de la méthode de Vigenère.
Cette première phase réalisée, il sera demandé de produire un programme « cassant » le message crypté.
Pour cela plusieurs méthodes pourront être utilisées : indice de coïncidence, force brute …
